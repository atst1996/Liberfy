# SocialApis(α)
WebサービスのAPIを使用するための.NET Framework向けのライブラリです。現在以下のサービスの対応作業中です（いくつか増やす予定です）。
- Twitter
- Mastodon

### 目的
現在Twitterのみに対応しているクライアントソフト『[Liberfy](https://github.com/atst1996/Liberfy)』を複数の他のサービスへ接続にも対応させることを主な目的としています。
